# Data Repository for the NDC Insights Series

This repository serves as the data repository for the the NDC Insights Series Digital Report on the [Data Futures Exchange](https://data.undp.org/insights/ndc-insights-series). It contains all the necessary datasets, geoJSON files, and resources required for the proper functioning and maintenance of the dashboard.

The main dashboard repository, which contains the source code and documentation for the dashboard, can be found here [**dv-ndc-insights-series**](https://github.com/UNDP-Data/dv-ndc-insights-series).
